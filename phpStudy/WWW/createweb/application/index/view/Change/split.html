# 输出文件 源文件 操作

步骤：

1. 点击“文件”或“链接”按钮切换本地文件或在线文件。点击“选择文件”按钮选择本地文件或输入在线文件URL。视频格式可以是MP4，AVI，FLV，MPG，MKV，WMV，M2TS，WEBM，ASF，MOV，M4V，RM，VOB或OGV。视频文件大小限制为200M。

2. 设置视频开始位置和持续时间。格式为：hh:mm:ss[.xxx]。例如：00:10:24。

3. 点击“提交”按钮提交表单。输出文件将会列在“输出结果”下面。点击 图标显示文件二维码或将文件保存到Google Drive或Dropbox。

关于：

您可以使用此视频在线分割程序来分割MP4文件，分割MKV文件，分割AVI文件，分割MPEG文件或分割WMV文件等。
